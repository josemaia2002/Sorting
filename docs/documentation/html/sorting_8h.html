Partitions the elements of an array based on a pivot value.

This function partitions the elements of an array specified by the range defined by the pointers 'first' and 'last' based on a pivot value. The elements smaller than the pivot are moved to the left side of the pivot, and the elements greater than or equal to the pivot are moved to the right side. The pivot element itself is also moved to its final sorted position. The function returns a pointer t o the pivot element's final position in the array.

Template Parameters

Parameters

lastPointer to the last element in the array (exclusive).

Returns
A pointer to the pivot element's final position in the array.
