Applies conditioned disorder to an array. 

This function applies conditioned disorder to the elements of an array specified by the range defined by the pointers \textquotesingle{}first\textquotesingle{} and \textquotesingle{}last\textquotesingle{}. The conditioned disorder is applied by swapping pairs of elements in the array. The number of pairs to be swapped is determined by the \textquotesingle{}porcent\textquotesingle{} parameter.
